Environment Setup
Create project folder and virtual environment

📁 Create Project Structure

bash 
mkdir StudyBuddy-RAG
cd StudyBuddy-RAG
mkdir .streamlit

🐍 Create Virtual Environment

Windows:

bash 
python -m venv venv
venv\Scripts\activate

Mac / Linux:

bash 
python3 -m venv venv
source venv/bin/activate

📦 Install Dependencies

bash 
pip install streamlit google-generativeai sentence-transformers faiss-cpu numpy pypdf
💡
The first run downloads the all-MiniLM-L6-v2 embedding model (~90 MB). Subsequent runs use the cached version.
2
Get Your Gemini API Key
Free tier available, no credit card needed
  1. Go to Google AI Studio

    Visit aistudio.google.com/app/apikey and sign in with your Google account.

  2. Create an API Key

    Click "Create API Key" → select a Google Cloud project or create a new one.

  3. Copy the key

    It looks like: AIzaSyABC123... — copy it immediately, you won't see it again.

  4. Save to secrets file

    Create .streamlit/secrets.toml and paste your key there.

toml 
# .streamlit/secrets.toml
GEMINI_API_KEY = "AIzaSyYourActualKeyHere"
⚠️
Never commit secrets.toml to Git. It's already in .gitignore — double-check before pushing.

Run Locally
Launch and test the app
bash

Streamlit opens http://localhost:8501 in your browser automatically.

🧪 Test Checklist

  • ✅ Upload a PDF from the sidebar — should show "X chunks indexed!"
  • ✅ Type a question in the Chat tab and get an AI answer
  • ✅ Click "Generate Summary" with and without a topic
  • ✅ Generate a 5-question Multiple Choice quiz
  • ✅ Create 10 flashcards and expand them
  • ✅ Adjust chunk size slider and re-upload — results should change

Deploy to Streamlit Cloud
Free hosting in under 5 minutes
  1. Push to GitHub

    Make sure .gitignore is in place so secrets aren't uploaded.

bash 
git init
git add .
git commit -m "StudyBuddy RAG initial commit"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
  1. Open share.streamlit.io

    Sign in with GitHub. Click "New app".

  2. Connect your repo

    Select your repo, branch main, and main file app.py.

  3. Add the secret

    Under "Advanced settings" → Secrets, paste:

toml 
GEMINI_API_KEY = "AIzaSyYourActualKeyHere"
  1. Click "Deploy!"

    Streamlit Cloud builds and serves your app at a public URL in ~2 minutes.

🎉
Your app is live! Share the URL with classmates. Streamlit Cloud's free tier gives you 1 GB RAM and unlimited public apps.

Next-Level Upgrades
Add these to make it internship-portfolio quality
  • 🔄 Multi-PDF Support

  • 🧠 Subject-Wise Memory

    Save FAISS indexes to disk with faiss.write_index() so re-uploads aren't needed.
